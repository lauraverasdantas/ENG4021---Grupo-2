# MOOD Flow 🌊🧠

> **O mapeamento inteligente das suas emoções.**

O **MOOD Flow** é uma plataforma de software voltada para o monitoramento contínuo da saúde mental, com foco no rastreamento de humor e na gestão da ansiedade. Nascido da necessidade de transformar percepções subjetivas em dados tangíveis, o aplicativo ajuda os usuários a identificar padrões comportamentais e emocionais ocultos em suas rotinas.

---

## 🎯 Visão do Produto

Muitas vezes, picos de ansiedade ou mudanças bruscas de humor parecem surgir do nada. A proposta do MOOD Flow é atuar como um diário inteligente que não apenas registra essas variações, mas analisa o histórico do usuário para prever gatilhos e evidenciar ciclos emocionais. É uma ferramenta tanto para o autoconhecimento quanto para o acompanhamento sistemático do bem-estar.

## ✨ Funcionalidades Principais

* **📝 Registro Diário de Humor:** Interface intuitiva com seleção de dia e mês para inputs rápidos sobre o estado emocional.
* **🔍 Monitoramento de Ansiedade:** Escalas de intensidade e registro de possíveis gatilhos para rastrear os níveis de estresse e ansiedade ao longo do tempo.
* **📊 Reconhecimento de Padrões (Analytics):** Cruzamento de dados históricos para identificar tendências (ex: "Sua ansiedade costuma aumentar nas noites de domingo").
* **👤 Gestão Completa de Perfil:** Sistema de registro, login autenticado, edição de dados do usuário e segurança das informações pessoais.

---

## 🛠️ Arquitetura e Tecnologias

O projeto foi estruturado para ser escalável, seguro e responsivo, utilizando as seguintes tecnologias:

* **Backend:** Python com o framework Django (arquitetura MVT - Model, View, Template).
* **Frontend:** HTML5, CSS3 e JavaScript puro para interfaces dinâmicas e responsivas.
* **Banco de Dados:** Sistemas relacionais baseados em SQL para armazenamento estruturado dos registros e perfis de usuário.
* **Lógica e Algoritmos:** Desenvolvimento focado em processamento de dados e identificação de padrões comportamentais.
